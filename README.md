# Guess the Number

A quick Shiny app that demonstrates basic functions of 
[shinyjs](https://github.com/daattali/shinyjs) and
[googlesheets](https://github.com/jennybc/googlesheets).

## Run it locally

```r
shiny::runGitHub("seankross/guess-the-number")
```
